# Privacy Policy — Knox Portal Obfuscator

_Last updated: 2026-06-16_

Knox Portal Obfuscator ("the extension") is designed to protect your privacy. This policy
explains exactly what the extension does and does not do with data.

## Data we collect

**None.** The extension does not collect, transmit, sell, or share any personal or sensitive
information. There are no analytics, no tracking, no advertising, and no remote servers.

## How the extension works

The extension runs only on the Samsung Knox admin portal (`*.samsungknox.com`). On those pages
it reads the page's visible text in order to detect sensitive values — such as license keys,
IMEI/MEID numbers, device serial numbers, email addresses, and Wi-Fi MAC addresses — and display
them in a masked form so they are not exposed during screen sharing.

All of this processing happens **locally in your browser**. The detected values are never sent
anywhere, never logged, and never stored. They exist only in the page you are already viewing.

## Settings storage

The extension stores a small set of **preferences** using your browser's built-in storage
(`storage.sync`): whether masking is currently on or off, and which categories of data you have
chosen to mask. These settings contain no personal data and no captured page content. They are
used solely to remember your configuration between sessions and, if your browser syncs
extension settings, may sync across your own signed-in browser profiles.

## Permissions

- **storage** — to save your on/off and masking preferences (described above).
- **host access to `*.samsungknox.com`** — so the extension can mask data on the Knox portal,
  and nowhere else. The extension cannot access any other website.

## Remote code

The extension contains no remote code. All code runs from files included in the published
package; nothing is downloaded or executed from external sources.

## Changes

If this policy changes, the "Last updated" date above will be revised.
